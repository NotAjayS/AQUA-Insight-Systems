##  Tech Stack
- HTML, CSS, JavaScript  
- Firebase (Hosting & Database)

---

##  Features
- Real-time monitoring of water levels  
- Dashboard for daily, monthly, and yearly insights  
- Alerts for overflow or low-water conditions  
- Modern responsive design for all devices
